Projeto realizado para a ingestão de dados de vendas para análise e projeções futuras para os negócios.

Temos 3 etapas nesse projeto, sendo elas chamadas respectivamente de camada BRONZE, SILVER e GOLD.

Nesso projeto utilizamos o Pyspark e Spark dentro do ambiente DATABRICKS community, abordagem de arquitetura medalhão para o tratamento do dados.
